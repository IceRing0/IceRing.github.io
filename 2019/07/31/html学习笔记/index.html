Html学习笔记

  • 2019-07-31

Html

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。

B/S架构

B是指浏览器

S是指服务器

网络概念

  • Http 超文本传输协议

  • url 统一资源定位符

  • uri 统一资源标识符

Html网页

  • 开发流程

    1. 编写网页

    2. 本地测试

    3. 部署

      将本地的html网页上传至云服务器上，并且部署到apache静态服务器上

    4. 网络测试

  • 网页元素

    由程序员编写，浏览器解析

    1. 注释

    2. 元素

        1. 开始标签，子元素，结束标签

    3. 属性

      • 核心属性

        <div class="first one">
    one
</div>

      • 特有属性

      • h5拓展属性

        <div data-nn='one'></div>

  • 结构

    1. doctype

      声明部分

    2. html

      • head

        不会显示到浏览器窗口中

        1. title

          显示在浏览器选项卡

        2. meta

          用于设置元信息，配置

      • body

        网页主体

  • 元素

    块元素

    作用：

    作为页面框架，或者容器，是页面的主体

    特性：

    1. 独占一行

    2. 默认宽度为100%，默认高度由子元素或者内容决定

    3. 可为其指定宽高

      style=”width:;height:;”

    div 无意义的块元素（无招）

    h1~h3 标题

    属性：

    1. font-size
    2. font-weight
    3. margin

    p 段落

    html

    body

    ul、li 无序列表，列表项

    1. ul

      margin

      padding

      list-style

    2. li

    ol、li 有序列表，列表项

    1. ol

      margin
      padding
      ist-style

    dl、dt、dd 有序列表

    1. dl

      margin

    2. dd

      margin

    h5新增的块元素 （语义化标签）

    1. header(头)
    2. nav
    3. article（主体）
    4. section (部分)
    5. footer（脚）
    6. aside

    行内元素

    作用

    点缀网页，填充内容。

    特性

    1. 与其他行内元素共享一行空间
    2. 默认宽高由内容决定
    3. 不能为其指定宽和高
    4. 行内元素中不可以嵌套块元素，但块元素中可以嵌套行内元素

    span 无意义的行内元素

    a 超级链接（axure中的open link）

    1. 默认样式

      1. color
      2. text-decoration
      3. cursor

    2. 默认事件（js - dom） 点击事件

      href（url、相对路径、绝对路径、、mailto、锚点）

    img

    1. src 图片地址
    2. url（相对路径、绝对路径、base64格式的数据）
    3. alt 图片找不到的时候文本替换

    功能元素

    表格

    table

    1. 【属性】border、width、、、、

    2. caption 表格标题

    3. thead 表头部分

      tr

      th/td

    4. tbody 表格主体部分【不可以省略】

      tr

      td/th【属性】colspan、rowspan、width、align

      td/th中可以放任意子元素

    5. tfoot 表底部

      tr

      td

    注意： 每一行中的列数应该是相等。

    表单（前后台交互）

    用户 -> 表单 -> 后台 -> 数据库

    form

    1. action 后台接口地址

    2. method 请求方式

      get

      • 参数拼接在url后面，通过?来分割。
      • 传递参数较少。
      • 查询学生信息、通过id删除学生信息。

      post

      • 参数存放在请求体中，安全。
      • 传递参数更多。
      • 保存或者更新学生信息、批量删除。

    3. enctype 编码方式

      1. enctype=’application/x-www-form-urlencoded’

        浏览器就会将参数转换为【查询字符串 qs】格式。

      2. multipart/form-data

        当有附件在表单中的时候，enctype务必设置为这种格式。

    4. ajax

    input

    1. name

      不能省略，作为参数中的key。

    2. value

      作为参数中的value,在按钮中务必指定value值。

    3. type

      text 单行文本框
      password 密码框
      submit 提交按钮
      file 附件选择器
      radio 单选按钮
      checkbox 复选按钮

    4. checked

    5. placeholder

    select

    1. option
    2. 标签中的文本显示在网页中，提交的值应该是option的value值，当这个值没有设定的时候，提交的是标签中的文本。
    第三方库
    1. 日期插件
    2. 地址选择器地址
    3. 附件上传
    4. 富文本
